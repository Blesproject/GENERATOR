# BLESS GENERATOR

Welcome To Bless microservice generator

## Install

```
pip3 install .
```

## Try

Create Your Project

```
mkdir ~/project1
cd ~/project1
```

Setup your pc connect to internet

```
bless create
```

Set up Object Yaml File

```
bless build
```

Run Your Project

```
bless run
```
